# Summary

Gehäuse für SPI SD-Kartenleser von Catalex
Housing for SPI SD card reader from Catalex

The Housing is snap-fit. It closes without screws or glue and holds the SD card reader securely in place.

Two versions included: 
- one with text "SD" on top part and connector indication on bottom part and
- one without any text
Both versions can be mixed, e.g. top with text + bottom without, as seen on the pictures.

For SPI SD card reader as found on:
http://www.dx.com/p/spi-microsd-card-adapter-v0-9b-for-arduino-works-with-official-arduino-board-246784
